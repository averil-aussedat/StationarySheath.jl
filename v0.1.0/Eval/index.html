StationarySheath.eval_fe! — Method
eval_fe!(res, f_eb, params, phix, v)

Return the value $f_e(x,v)$.

Note that the non-emitting boundary condition has precedence over the prescribed value $f_{e,b}$, and the result will be 0 whenever 𝓛ₑ(x,v) ⩾ 𝓛ₑ(1,0).
